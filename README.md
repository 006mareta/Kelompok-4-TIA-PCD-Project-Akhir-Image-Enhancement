Mata Kuliah : Pengolahan Citra Digital

Project ini membahas implementasi metode Image Enhancement menggunakan kombinasi Gaussian Filter dan Histogram Equalization untuk meningkatkan kualitas citra Dental X-Ray. Metode Gaussian Filter digunakan untuk mengurangi noise pada citra, sedangkan Histogram Equalization digunakan untuk meningkatkan kontras sehingga detail pada citra X-Ray dapat terlihat lebih jelas.
